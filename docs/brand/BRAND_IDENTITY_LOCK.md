# FlyStay Brand Identity Lock

## Overview

FlyStay is a luxury travel booking platform targeting the Saudi and Gulf markets. This document defines the locked brand identity.

---

## Brand Identity (LOCKED)

| Property | Value |
|----------|-------|
| Brand Style | Luxury Travel Minimal |
| Primary Color | Charcoal `#2B2D31` |
| Accent Color | Champagne `#CDB68B` |
| Background | Ivory `#F8F6F1` |
| Secondary Surface | Sand `#E7E3DC` |
| Soft Border | Mist `#F1F1F1` |
| English Typeface | Playfair Display |
| Arabic Typeface | Cairo |
| UI Direction | RTL Arabic First |
| Brand Feeling | فاخر، هادئ، عالمي، نظيف |

---

## Color Palette

### Primary Colors
- **Charcoal** `#2B2D31` - Buttons, headers, dark backgrounds
- **Champagne** `#CDB68B` - Luxury touches, accents, highlights

### Background Colors
- **Ivory** `#F8F6F1` - Page background
- **Sand** `#E7E3DC` - Cards, secondary elements
- **Mist** `#F1F1F1` - Borders, dividers

### Text Colors
- Primary: `#2B2D31`
- Secondary: `#5C5D60`
- Muted: `#9A9A9C`

### Status Colors
- Success: `#4A7C59`
- Error: `#C45C5C`
- Warning: `#D4A84B`
- Info: `#5B7B9A`

---

## Typography

### Arabic - Cairo
```css
font-family: 'Cairo', sans-serif;
font-weights: 300, 400, 500, 600, 700;
direction: rtl;
```

### English - Playfair Display
```css
font-family: 'Playfair Display', serif;
font-weights: 400, 500, 600, 700;
```
